# 🏥 Hospital Management System

A modern full-stack Hospital Management System built with **React, TypeScript, Vite, Firebase, and Tailwind CSS**. The application provides an intuitive interface for patients to book appointments and administrators to manage doctors and appointments efficiently.

---

## ✨ Features

### 👨‍⚕️ Patient
- View hospital information
- Browse departments
- View doctors
- Book appointments
- Responsive UI for desktop and mobile

### 🔐 Admin
- Secure Admin Login
- View all appointments
- Manage appointment requests
- Dashboard for hospital administration

### 📱 User Interface
- Modern responsive design
- Smooth animations
- Fast loading with Vite
- Professional hospital-themed UI

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/hospital-management-system.git
```

Move into the project

```bash
cd hospital-management-system
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory.

Example:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
